JVector Project Page

JVector is an open source vector image editing program distributed for free. It runs on numerous platforms, with the officially supported ones being Windows 2000 & XP, Linux (Kernel 2.4 and higher), and Mac OS X.

JVector is written in the Java Programming Language. It is an open source project distributed under a GNU General Public Licence (GPL). It aims to provide a moderately advnaced and featured program, but with a great and friendly GUI, and an ease of use equal to basic paint programs, such as MS Paint.
